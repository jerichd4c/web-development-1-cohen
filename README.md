## About The Repository

This repository brings together the material covered in the **Web Development 1** course. Its purpose is to keep class practices, guided exercises, and projects built with **HTML**, **CSS**, and **JavaScript** in one organized place.

It works as a learning log: each folder represents a different activity and makes it easy to review the course progress in a clear and structured way.

## Getting Started

To open any practice, no extra dependencies are required. You only need a modern browser and, in some cases, a local server to test modules, audio, or browser storage.

### Prerequisites

* A modern web browser.
* Visual Studio Code or a similar editor.
* Optionally, an extension such as Live Server to run HTML files locally.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/jerichd4c/web-development-1.git
   ```
2. Open the folder for the exercise or project you want to review.
3. Run the main HTML file in your browser or with Live Server.

## Repository Structure

These are the practices and projects currently available in the repository:

* `carousel component/`: carousel component built with HTML, CSS, and JavaScript.
* `collision basic example/`: basic collision example between elements.
* `geolocator/`: geolocation practice using browser APIs.
* `music player/`: music player with modular interface and local persistence.
* `space invaders clone/`: playable Space Invaders clone built with vanilla JavaScript.
* `spritesheet animation test/`: animation tests using sprites and canvas.

## Main Projects

These are some of the most important works developed during the course. Each one has its own internal documentation when applicable.

### [Space Invaders Clone](space%20invaders%20clone/README.md)
An arcade classic clone developed with vanilla JavaScript, custom sprites, and original gameplay logic.
* **Features**: player movement, shooting, enemies, collisions, progressive difficulty, and sound.
* **Documentation**: [Project README](space%20invaders%20clone/README.md)

### [Music Player](music%20player/README.md)
A local audio player with modular playlist management and browser persistence.
* **Features**: playback, playlists, local storage, and separated components.
* **Documentation**: [Project README](music%20player/README.md)
